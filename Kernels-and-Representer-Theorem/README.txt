Kernels and the Representer Theorem — a teaching tutorial
=========================================================
Contents
  Kernels-and-Representer-Theorem.md  the tutorial (figures linked from ./figures/)
  figures/                            all 9 figures (PNG)
  make_figures.py                     the Python script that generated every figure

To READ it with images + rendered equations, open the .md in a Markdown viewer
that supports LaTeX math and local images — e.g. Obsidian, Typora, or VS Code with
a "Markdown+Math" extension. Keep the .md and the figures/ folder together.

Note: opening the .md in a plain preview that doesn't resolve local image paths
(or doesn't render $...$ math) will show blank images / raw LaTeX. That's a viewer
limitation, not a problem with the file. Ask for an HTML version if you'd prefer a
single file that renders anywhere in a browser.

The math has been checked: all 258 equations parse in KaTeX, and the key results
(kernel-trick identity, spike norm, sine roughness, kernel-ridge solution) were
verified numerically.
