Kernel-to-Regularization Tutorial (editable Markdown bundle)
------------------------------------------------------------
- Kernel-to-Regularization-Tutorial.md : the tutorial, with figures linked from ./figures/
- figures/*.png                        : all 13 figures (generated with Python/matplotlib)

To see images + rendered math, open the .md in a Markdown editor that supports
LaTeX math and local images, e.g. Obsidian, Typora, or VS Code (with a
"Markdown+Math" / "Markdown Preview Enhanced" extension). Keep the .md file and
the figures/ folder together.

Prefer to just read it? Open the standalone .html version instead — it renders
everywhere (any web browser) with no setup.
